# Smart-Doors
A project based on Raspberry Pi, designed to allow the door to be opened using face recognition.

Używając motherboard podłączenie pinów:
1. Zielona dioda led = 5
2. Czerwona dioda led = 4
3. Buzzer = 17
4. pinBtn=13
5. Elektromages =6
6. pinSensor=19
7. Ekran LCD:
	1. 
	2.
